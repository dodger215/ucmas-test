API Documentation: Update User Score

This API allows you to update the user's score for a specific level and stage.

API Endpoint

URL: http://localhost/project/math_test/update/beginning_level/add/

Method: POST

Content-Type: application/json

Request

The API expects a JSON body containing the user's ID, level, stage, and score.

{
    "id": 123,
    "level": "level 1",
    "stage": "Stage 1",
    "score": 85
}

Explanation:

  • id: The unique ID of the user (e.g., 123).
  • level: The level name (e.g., "level 1").
  • stage: The specific stage within the level (e.g., "Stage 1").
  • score: The score to be updated (e.g., 85).

Response

The API will respond with a JSON object indicating the success or failure of the operation.

{
    "status": "success",
    "message": "User score updated successfully."
}

Possible responses:

  • status: "success" or "error".
  • message: A message explaining the result of the request.

PowerShell Example Command

To make a request to the API, you can use the following PowerShell command:

Invoke-RestMethod -Uri "http://localhost/project/math_test/update/beginning_level/add/" -Method Post -Body '{"id":123,"level":"level 1","stage":"Stage 1","score":85}' -ContentType "application/json"
